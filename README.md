# ML-for-Car-Diagnostic-Agent-AI-Assistant

Overview
The ML Car Diagnostic Agent AI Assistant is an advanced machine learning system designed to revolutionize automotive repair diagnostics. This AI-powered solution analyzes historical repair data to provide data-driven insights, predict repair success probabilities, and optimize automotive service operations.

Features
Predictive Diagnostics: Analyzes patterns across 10,000+ automotive repair cases

Success Probability Estimation: Predicts repair outcomes before work begins

Brand-Specific Insights: Identifies common failure patterns across different vehicle manufacturers

Real-time Processing: Provides immediate diagnostic recommendations

Multi-class Classification: Categorizes problems into specific automotive systems

Severity Assessment: Prioritizes repairs based on impact levels

Dataset Information
The dataset contains 10,000 real-world automotive diagnostic cases with 12 comprehensive features:

Dataset Structure
Record ID: Unique identifier for each case

Car Name: Complete vehicle identification (brand, model, year)

Problem Classification: Category of automotive issue (Engine, Electrical, Transmission, Brakes, Battery)

Problem Description: Specific symptoms and issues reported

Severity: Problem impact level (Low, Medium, High)

Diagnosis: Technical assessment of root cause

How to Fix the Problem: Recommended repair procedure

Solution Used: Actual solution implemented

ECU Data: Error codes and diagnostic information

Service History: Vehicle maintenance background

Repair Status: Final outcome (Fixed, Not Fixed, In Progress)

Results: Resolution status (Resolved, Unresolved, Monitoring)

Vehicle Coverage
Brands: Toyota, Honda, Ford, Chevrolet, Hyundai, Kia, Nissan, BMW, Mercedes

Models: 10 popular models per brand

Years: 2015-2025 vehicles
